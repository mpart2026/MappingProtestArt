# Mapping Protest Art

Mapping Protest Art is a project that collects and visualizes protest art from around the world, spanning murals, installations, digital works, and ephemeral street actions, to reveal shared strategies, local struggles, and global solidarities.

This project was created by Lini Radhakrishnan as a capstone project to fulfill the Masters Degree requirement for the Digital Humanities program at the Graduate Center, CUNY.
